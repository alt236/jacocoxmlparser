# Jacoco XML Parser


```
usage: jacocoxmlparser-1.0-SNAPSHOT.jar -g | -p -i <arg>
version: 1.0-SNAPSHOT
Parse Jacoco XML files and print stats.

 -g,--global        If set, print global coverage statistics
 -i,--input <arg>   The target XML file
 -p,--package       If set, print package coverage statistics

Sample usage:

 * jacocoxmlparser-1.0-SNAPSHOT.jar --input ~/tmp/file.xml -g

```
